# 1. Introduction

The Universal Smart Gateway (USG) is an ESP32-based Internet of Things (IoT) system designed to automate monitoring and control of agricultural and industrial environments.

The system collects data from multiple sensors, displays information on an OLED screen, hosts a web-based SCADA dashboard, and automatically controls irrigation using a relay.

---

# 2. Problem Statement

Many farms and small industries still rely on manual monitoring and control of equipment.

This leads to:

- Water wastage
- High labour costs
- Delayed response to equipment failures
- Poor monitoring of environmental conditions

A low-cost intelligent monitoring system is therefore required.

---

# 3. Project Aim

To design and develop a low-cost Universal Smart Gateway capable of monitoring sensors, displaying real-time information, and automating irrigation using ESP32 technology.

---

# 4. Objectives

- Develop an ESP32-based monitoring system.
- Display sensor information on an OLED display.
- Build a Wi-Fi web dashboard.
- Automate irrigation using soil moisture readings.
- Monitor voltage and current.
- Support future LoRa communication.
- Provide a SCADA-style interface.
