## Developer

## Overview
The Universal Smart Gateway (USG) is an ESP32-based IoT platform designed for smart irrigation, industrial monitoring, and automation. It combines sensors, OLED graphics, Wi-Fi networking, and a web-based SCADA dashboard.

## Features
- ESP32 Heltec WiFi LoRa 32 V3
- OLED graphical display
- Smart irrigation control
- Soil moisture monitoring
- Wi-Fi web dashboard
- Relay automation
- Real-time clock (NTP)
- Voltage and current monitoring
- LoRa-ready architecture
- SCADA-style interface

## Hardware
- Heltec WiFi LoRa 32 V3
- OLED Display
- Soil Moisture Sensor
- Relay Module
- DHT Sensor
- Voltage Sensor
- Current Sensor
- LDR Sensors
- Servo Motors
